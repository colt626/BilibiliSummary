# BilibiliSummary
A chrome extension helps you summary video on bilibili.

B站视频总结：视频太长不看了给你节约点时间。

# 使用方法
在另一个页面登录你的chatgpt账号，然后刷新页面，就能使用。

使用过于频繁，有可能被封号，好自为之。
## 特性
点击总结，使用chatgpt总结.

## 构建
1. 安装nodejs
2. 运行npm install
3. 运行 npm run fast-build
4. 将dist文件夹加载到chrome扩展中
